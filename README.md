# JobMiw

JobMiw is a lightweight, privacy-focused, and free open-source job application tracker built as a Progressive Web App (PWA). It works entirely on your device with no ads, subscriptions, accounts, or tracking.

The goal of JobMiw is simple: provide a straightforward way to track job applications, interviews, and prep without the clutter, paywalls, and bloat commonly found in other job-hunting tools.

## Features

* Free and open source (MIT License)
* Works offline
* No account required
* No ads or subscriptions
* Privacy-focused local storage
* Application tracking with status, notes, and history
* Built-in AI Interview Trainer
* Job preparation checklist
* Multiple color themes (Purple, Blue, Red/Pink, Dark)
* Lightweight and fast
* Installable on Android, iOS, Windows, macOS, and Linux
* Automatic updates when using the hosted version

## Installation and Usage Guide

### Use the Hosted Version

The easiest way to use JobMiw is through the hosted web version.

Open the application in your preferred browser and start tracking immediately. All data is stored locally on your device and is never sent to any server.

### Install on Android

To install JobMiw as a PWA on Android:

1. Open JobMiw in Chrome or Brave.
2. Tap the browser menu.
3. Select **Install App** or **Add to Home Screen**.
4. Confirm the installation.

### Install on iPhone and iPad

To install JobMiw on iOS:

1. Open JobMiw in Safari.
2. Tap the **Share** button.
3. Select **Add to Home Screen**.
4. Tap **Add**.

### Install on Desktop

To install JobMiw on Windows, macOS, or Linux:

1. Open JobMiw in Chrome, Edge, or Brave.
2. Click the install icon in the address bar.
3. Confirm installation.

### Run Locally

If you prefer running JobMiw locally:

1. Download the latest release ZIP file.
2. Extract the contents.
3. Open `index.html` in any modern browser.

No build process or additional dependencies are required.

## Updating

### Hosted Version

The hosted version automatically receives updates whenever a new version is released.

### Local Version

If you are using the downloaded ZIP version, you will need to manually download future releases to receive updates.

## Data Privacy

JobMiw uses a local-first approach.

* No user accounts
* No analytics
* No tracking
* No cloud storage
* No personal data collection

All information remains stored locally on your device.
